# Task Master Pro

A simple, functional To-Do List application built as a classic introductory project for front-end web development.

## Project Icon

The application uses the following custom icon for its title bar branding (favicon):

![Application Logo](/assets/logo.jpg)


## Tech Stack

This project is built using the core pillars of front-end web development:

* **HTML5:** Defines the structure and semantic content of the task list.
* **CSS3:** Provides modern, clean styling, including interactive states.
* **JavaScript (Vanilla):** Handles all the application logic, DOM manipulation, and event management.

## Features

* **Add Tasks:** Enter text and click 'Add' or press Enter to add a new item.
* **Mark Complete:** Click the text of any task to toggle a visual strikethrough.
* **Delete Tasks:** Click the 'Delete' button adjacent to any task to remove it permanently.

## Setup and Installation

This is a plain front-end project; no compilation or server installation is required.

1.  **Download/Clone:** Download the project files (`index.html`, `style.css`, `script.js`, and `logo.png`) into a single folder.
2.  **Open:** Double-click the `index.html` file to open it in your default web browser.
